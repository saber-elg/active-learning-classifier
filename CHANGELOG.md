## [1.0.0] - 2024-02-01

### Added
- Initial release of Active Learning Classifier
- ResNet-style CNN architecture with skip connections
- Multiple query strategies (Uncertainty, Margin, Entropy, BALD)
- Interactive Streamlit web application
- Comprehensive metrics tracking and visualization
- Data augmentation pipeline
- Benchmark comparison tools
- Complete test suite
- CI/CD pipeline with GitHub Actions
- Docker support with docker-compose
- Extensive documentation (README, ARCHITECTURE, API, CONTRIBUTING)
- Professional project structure
